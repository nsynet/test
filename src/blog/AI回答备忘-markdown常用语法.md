Markdown 是一种轻量级标记语言，它的核心语法简洁直观，让你能专注于内容本身。不过，标准的 Markdown 语法并不直接支持调整图片大小。要实现这个需求，通常需要借助 HTML 标签。

### 如何调整图片大小（以 50% 比例显示为例）

由于标准的 Markdown 图片语法无法直接设置尺寸，最通用和灵活的方法是使用 HTML 的 `<img>` 标签。你可以通过几种方式将图片大小调整为原来的 50%。

#### 方法一：使用 `style="zoom:50%"` (推荐)

这是最直接、语义最清晰的方法，`zoom` 属性表示相对于图片原始尺寸的缩放比例。

```html
<img src="图片路径/图片文件名.jpg" alt="图片描述" style="zoom:50%" />
```
> **说明**：`alt` 属性中的文本用于在图片无法加载时显示，也有助于无障碍阅读。

#### 方法二：使用 `width="50%"`

这种方法将图片宽度设置为父容器宽度的 50%，在某些布局下可能不会严格遵循图片的原始宽高比。但如果你希望图片能根据容器大小灵活缩放，这个方法很有用。

```html
<img src="图片路径/图片文件名.jpg" alt="图片描述" width="50%" />
```

#### 方法三：同时设置宽度和高度百分比

你也可以同时指定宽高百分比，但如果不锁定宽高比，图片可能会变形。

```html
<img src="图片路径/图片文件名.jpg" alt="图片描述" width="50%" height="50%" />
```

#### 补充：同时调整图片位置

你还可以结合 HTML 的 `<div>` 标签来调整图片的对齐方式。

```html
<div align="center">
  <img src="图片路径/图片文件名.jpg" alt="图片描述" style="zoom:50%" />
</div>
```
> 这段代码会将缩放到 50% 的图片在其容器中居中显示。

如果你使用的是某些特定的 Markdown 编辑器或平台（如 Typora、GitHub），它们可能有自己专属的调整语法，但使用 HTML 标签是最通用、最可靠的解决方案。
